# ❤️ Projeto: Declaração de Amor Interativa

Este projeto foi desenvolvido como uma aplicação prática dos conceitos aprendidos na disciplina de **Desenvolvimento Front-End** na **Unisenai-PR**. O objetivo foi criar uma página web responsiva, interativa e visualmente atraente, utilizando as tecnologias fundamentais da web.

## 🚀 Tecnologias Utilizadas

O projeto demonstra o domínio das seguintes camadas de desenvolvimento:

* **HTML5:** Estruturação semântica do conteúdo, incluindo a integração de elementos de mídia (`<audio>`) e de desenho gráfico (`<canvas>`).
* **CSS3:** Estilização avançada, utilizando posicionamento, animações flutuantes para imagens e layouts modernos.
* **JavaScript (Vanilla):** Lógica de programação aplicada para manipulação do DOM e criação de animações baseadas em física básica.

---

## 🧠 Aprendizados Aplicados

Durante o desenvolvimento, foram colocados em prática os seguintes tópicos abordados em sala de aula:

### 1. Manipulação de Elementos Gráficos com Canvas
Utilizei a **API Canvas** para renderizar corações animados em tempo real. Isso envolveu o conhecimento de:
* Contexto 2D (`getContext('2d')`).
* Criação de caminhos complexos com curvas de Bézier (`bezierCurveTo`).
* Ciclos de animação com `requestAnimationFrame` para garantir performance e fluidez.

### 2. Programação Orientada a Objetos (POO) no JS
A implementação dos corações foi feita utilizando **Classes**, permitindo que cada "coração" no fundo seja um objeto independente com propriedades de velocidade, tamanho, opacidade e movimento (oscilação senoidal) próprias.

### 3. Interatividade e Eventos
* Implementação de controles de áudio customizados via JavaScript para contornar restrições de *autoplay* dos navegadores modernos.
* Tratamento de eventos de redimensionamento (`resize`) para garantir que o projeto seja responsivo e o canvas ocupe sempre a tela inteira.

### 4. Estilização e UX
* Uso de animações CSS para criar o efeito de "fotos flutuantes".
* Design focado na experiência do usuário (UX), com feedback visual imediato ao interagir com os controles de música.

---

## 🎨 Demonstração
O projeto consiste em uma página onde corações sobem suavemente pelo fundo da tela enquanto fotos do casal flutuam em destaque. Ao clicar no botão interativo, a trilha sonora do casal é reproduzida, completando a experiência sensorial.
